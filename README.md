# 📝 Notes App Frontend (Next.js)

This is the frontend for the Notes App.  
It communicates with the FastAPI backend to manage notes.

---

## 🚀 Features
- Login with username/password
- Fetch notes by user
- Create new notes
- (Optional: Update & Delete)

---

## ⚙️ Setup

### 1. Install dependencies
```bash
npm install
```
